putBulk(bucketName, fileInfoList)¶

Initiates a start bulk put with the remote ds3 endpoint. The fileInfoList is a list of (objectName, size) tuples. objectName does not have to be the actual name on the local file system, but it will be the name that you must initiate a single object put to later.

putJob(jobId)¶
putObject(bucketName, objectName, offset, size, jobId, realFileName=None, metadata=None)¶

Puts an object to the ds3 endpoint. Use realFileName when the objectName that you are putting to ds3 does not match what is on the local filesystem. Use metadata to set the metadata for the object. metadata’s value should be a dictionary, where keys are associated with either a value or a list of the values for that key.
